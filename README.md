# 💰 LoanApp – Employee Loan Management System

LoanApp is a web-based application built using **ASP.NET Core MVC** to manage internal employee loans, track transactions, and calculate balances efficiently.

---

## 🚀 Features

### 👥 Employee Management
- Add, edit, and manage employees
- Store department, phone, and employee code

### 💸 Loan & Transactions
- Record lending between employees
- Track all financial transactions
- Maintain full history

### 📊 Balance System
- Automatically calculate each employee’s balance
- Show who owes and who is owed

### 🔐 Role-Based Access
- **Admin**
  - Full access to all data
- **Employee**
  - Can only see:
    - Their own profile
    - Their own transactions
    - Their own balance

### 🖥️ User Interface
- Clean and responsive UI
- Built with Razor Views
- Arabic-friendly layout

---

## 🛠️ Technologies Used

- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  
- C#  
- Razor Views
